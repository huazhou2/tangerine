## ⚡ Quick Start (3 Commands)

### 1️⃣ Generate Splits for All 5 Folds
```bash
bash run_5fold_cv.sh \
    /gpfs/data/tsirigoslab/home/zhouh05/lung_ct/lungct_with_mrn_anonacc.csv \
    /gpfs/data/tsirigoslab/home/zhouh05/lung_ct/data/images_3d_swine \
    .
```
**Time**: ~5 minutes  
**Output**: `dataset_splits_fold0/` through `dataset_splits_fold4/`

### 2️⃣ Generate Training Scripts
```bash
python generate_fold_scripts.py
```
**Time**: < 1 second  
**Output**: `train_fold1.sh` through `train_fold4.sh` (fold0 already exists)

### 3️⃣ Submit All Training Jobs
```bash
for i in {0..4}; do sbatch train_fold$i.sh; done
```
**Time**: ~2-4 hours per fold (or ~2-4 hours if running in parallel on 5 GPUs)  
**Output**: `outputs/fold*_gamma2_pr_auc_TIMESTAMP/` directories

### 4️⃣ Collect All Results
```bash
python collect_5fold_predictions.py
```
**Time**: ~1 minute  
**Output**: `5fold_combined_test_predictions.csv` + `5fold_summary.json`

## 🔍 What Each Script Does

### `prepare_survival_dataset.py` (Modified)
**NEW PARAMETER**: `--fold_idx` (0-4, or None for single split)

```bash
# For fold 0
python prepare_survival_dataset.py \
    --metadata_csv /path/to/metadata.csv \
    --images_dir /path/to/images \
    --output_dir ./dataset_splits_fold0 \
    --fold_idx 0 \
    --seed 42

# For single split (old behavior)
python prepare_survival_dataset.py \
    --metadata_csv /path/to/metadata.csv \
    --images_dir /path/to/images \
    --output_dir ./dataset_splits \
    --seed 42
```

**What it does**:
- Uses StratifiedKFold to create 5 patient groups
- For fold i: uses fold i as test (20%), remaining 80% split into train (70%) and val (10%)
- Stratifies by cancer status
- Ensures no patient appears in multiple folds
- Outputs: `train.csv`, `val.csv`, `test.csv`, `config.json`

### `train_fold0.sh` (Template)
SLURM job script for training fold 0

```bash
sbatch train_fold0.sh
```

**What it does**:
- Checks if `dataset_splits_fold0/` exists (auto-regenerates if missing)
- Verifies no patient overlap between splits
- Trains model on fold 0 training data
- Evaluates on fold 0 test data (20% of all patients)
- Outputs to: `outputs/fold0_gamma2_pr_auc_TIMESTAMP/`

### `generate_fold_scripts.py`
Generates `train_fold1.sh` through `train_fold4.sh` from template

```bash
python generate_fold_scripts.py
```

**What it does**:
- Reads `train_fold0.sh`
- Replaces all fold indices (0 → 1, 2, 3, 4)
- Creates executable scripts for remaining folds
- Makes them all SBATCH-ready

### `run_5fold_cv.sh`
Generate all 5 splits in one go

```bash
bash run_5fold_cv.sh /path/to/metadata.csv /path/to/images .
```

**What it does**:
- Calls `prepare_survival_dataset.py` 5 times with fold_idx=0,1,2,3,4
- Creates `dataset_splits_fold0/` through `dataset_splits_fold4/`
- Verifies each split for data leakage
- Prints summary of patient distributions

### `collect_5fold_predictions.py`
Aggregate predictions from all 5 fold runs

```bash
python collect_5fold_predictions.py
```

**What it does**:
- Finds all `test_predictions.csv` files from fold0-4 outputs
- Combines them into one file
- Verifies ~20% of patients per fold
- Prints per-fold statistics
- Outputs:
  - `5fold_combined_test_predictions.csv` (all patient predictions)
  - `5fold_summary.json` (summary statistics)

## 🎯 Key Parameters (All Folds)

All folds use the same hyperparameters:

```
Epochs:           320
Patience:         100
Warmup epochs:    30
Batch size:       4
Grad accum:       4 (effective batch = 16)

Learning rates:
  Head (first 30 epochs):     1e-4
  Encoder (after warmup):     5e-6 (ratio=0.05)
  LLRD decay:                 0.75

Loss:
  Type:     Focal Loss + per-year weighting
  Gamma:    2.0
  Alpha:    0.25

Optimization:
  Optimizer:    AdamW
  Weight decay: 1e-3
  AMP:          Yes
  Augmentation: Yes

Random seed: 42 (ensures reproducibility)
```

## 🔄 Troubleshooting

### Common Issues

**Issue**: "prepare_survival_dataset.py: error: unrecognized arguments: --fold_idx"
```
Cause: Old version of script
Fix:   Ensure you're using the modified version from 5fold/
```

**Issue**: "StratifiedKFold not found"
```
Cause: sklearn version too old
Fix:   pip install --upgrade scikit-learn
```

**Issue**: Jobs fail with OOM (out of memory)
```
Fix:   Reduce batch size in train_fold*.sh
       --batch_size 2
```

**Issue**: test_predictions.csv not found
```
Cause: Training failed or didn't complete
Fix:   Check logs for errors: tail logs/tangerine_fold*_JOBID.out
```
